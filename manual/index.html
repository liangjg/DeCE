Introduction

General Information

DeCE is an ENDF-6 formatted file manipulation program, similar to STANEF by C.L. Dunford et al. (BNL), or FIXUP in PREPRO by D.E. Cullen (IAEA), but probably the closest to CRECTJ6 by T. Nakagawa (JAEA), as the calculations are performed interactively.

In this manual, the word ENDF-6 file is often used. This does not refer to a particular name of libraries, like ENDF/B-VI, but a data file which is in the ENDF-6 format in general.

In a general case, one data file can contain more than one materials, each of the materials specified by some different MAT numbers. However, DeCE assumes that an input ENDF-6 file includes only one MAT number.

DeCE does the following things. Here the word "section" means a data block specified by a combination of MF and MT numbers.

  • extract one section from a file, and display it
  • calculate add/sub/etc of two sections
  • manipulate a data-point in a section (replace, scaling, delete)
  • read data in an external file, and convert them into ENDF-6
  • reconstruct pointwise cross sections from resonance parameters
  • renumber all the sections, make a new dictionary, update NWD and NXC, then write the entire data on a new file.

DeCE is written in C++, and it has classes, ENDF and ENDFDict, which provide interface to the ENDF-6 specified data structure. The ENDF class is so general that it can be used in other user utility programs. This package contains some examples to use the ENDF class.

History

Starting in Dec. 2009, DeCE still has a short history in development. Some of major updates have a code name like, Talc, Gypsum, and Calcite. They are known to as "the Mohs scale of mineral hardness." The code was originally developed as E-Type, which was a converter program from the Hauser-Feshbach model code output into the ENDF-6 format, first appeared in Summer 2009. This was totally superseded by DeCE. The current version, built in Sept. 2015, is Adularia.

All versions up to DeCE (Fluorite) were just for the author's private use. Apatite is the first version of outside release (yet limited to the lab internal use). This version comes with some documents. Turquoise is a stable version of Apatite, including many bug fixes. Adularia, which was built in 2015, has several upgrades in the resonance region. DeCE was released as an open source software in 2016.
