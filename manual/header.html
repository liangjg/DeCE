Header Manipulation

showheaders (print parameters in header)

The first 4 lines in a ENDF file contain information on identification, version, and some general properties of the file. The showheaders command prints these values.

editheader (modify header parameter)

To change individual parameter, use editheader followed by the parameter name and value. When the version number NVER is set to 10,

   editheader NVER 10

The parameter names are NLIB, LREL, NVER, NSUB, NMOD, LDRV, NFOR, LRP, LFI, EMAX, TEMP, ELIS, STA, LIS, and LISO. See ENDF-6 manual for the meaning. ZA, AWR, and AWI are not allowed to change.
